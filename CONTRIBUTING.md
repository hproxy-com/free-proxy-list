# Contributing

This free proxy list is generated automatically and verified continuously, so we
do not accept raw proxy IPs by pull request: they would be overwritten on the
next hourly update. The useful ways to contribute:

## Suggest a source
Found a public proxy source we are not pulling from yet? Open an issue with the
link and we will add it to the verification engine. New sources flow into the
list automatically once they pass checks.

## Report a bad entry
If an IP listed here is yours and you want it removed, open an issue or email the
address on https://hproxy.com.

Thanks for helping keep the list fresh.
